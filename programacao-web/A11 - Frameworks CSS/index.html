Guia Bootstrap

Detalhes importantes a serem observados neste arquivos:

  1. navbar utilizado para navegação;
  2. container-fluid é um tipo de container especial, faz com que o container preencha toda a largura da viewport;
  3. container é o container normal, tem largura específica para cada resolução;
  4. row é a definição de uma linha. Linhas podem ter colunas. Os elementos podem ser aninhados infinitamente;
  5. col-lg-x é a definição de quantas colunas serão ocupadas no tamanho de tela large. x define quantas colunas serão ocupadas;
  6. col-md-x é a definição de quantas colunas serão ocupadas no tamanho de tela medium;
  7. col-sm-x é a definição de quantas colunas serão ocupadas no tamanho de tela small;
  8. col-xs-x é a definição de quantas colunas serão ocupadas no tamanho de tela extra-small;
  9. ml-auto utiliza flex-box para distribuir as colunas igualmente; ml: margin-left
  10. mr-auto utiliza flex-box para distribuir as colunas igualmente; ml: margin-right
  11. mx-auto utiliza flex-box para centralizar um elemento
  12. mais sobre os espaçamentos: Veja Aqui
